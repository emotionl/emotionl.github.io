# 使用nvm管理Nodejs

1. 安装脚本（[脚本地址](https://github.com/nvm-sh/nvm#install--update-script)）

   ```bash
   wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.2/install.sh | bash
   ```

2. 查看`Nodejs`版本

   ```bash
   nvm ls-remote
   ```

3. 安装 `Nodejs`

   ```bash
   nvm install v12.14.0
   ```

4. 测试`Nodejs`是否安装成功

   ```bash
   node -v
   npm -v
   ```
