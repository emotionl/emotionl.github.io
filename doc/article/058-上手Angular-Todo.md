## 什么是组件

模板 + 样式 + 类 + 元数据 = 组件

### 模板（试图布局）

- HTML
- 绑定和指令

### 样式（视图样式）

- CSS / SCSS / LESS

### 类 （控制试图）

- TS
- 属性：数据
- 方法：逻辑

### 元数据

- ng 需要的数据或配置
- 使用注解进行定义
