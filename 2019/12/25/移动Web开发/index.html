rem 布局

rem：font size of the root element

  • 字体单位：值根据html根元素大小而定，同样可以作为宽度高度等单位
  • 适配原理：将px替换成rem，动态修改html的font-size适配
  • 兼容性：IOS 6以上和Android 2.1以上，基本覆盖所有流行的手机系统
